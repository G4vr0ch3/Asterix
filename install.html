Asterix Installation

To install Asterix, first get a copy of the latest release here.
We would recommand that you start with the configuration of the Analysis Center Virtual Machine. To get an example of configuration, refer to the documentation.

Unzip the release to extract the source code:
~$ unzip Asterix-{RELEASE TAG}.zip

Select the appropriate setup file (> v0.0.3-ALPHA) and run it using python:
~/Asterix-{RELEASE TAG} $ python setup.py

Follow the displayed instructions, select your options.
You're done ! Reboot and run the software as the asterix user:
~$ python Asterix.py

Thank you for using Asterix ❤️
